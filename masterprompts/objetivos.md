# Product Objectives

El producto debe cumplir los siguientes objetivos:

1.
Crear cuentas en menos de un minuto.

2.
Permitir crear un catálogo profesional.

3.
Generar automáticamente un subdominio.

4.
Permitir compartir el catálogo inmediatamente.

5.
Administrar productos desde un panel sencillo.

6.
Escalar posteriormente hacia:

- CRM
- WhatsApp
- IA
- Automatizaciones
- Analytics

Todo sin modificar la experiencia inicial.

# Primary KPI

Tiempo promedio desde registro hasta publicación.

Objetivo:

Menos de cinco minutos.
