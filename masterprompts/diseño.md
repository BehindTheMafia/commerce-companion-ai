# Design System

Todos los componentes deben compartir exactamente el mismo lenguaje visual.

## Buttons

Radio consistente.

Sombras suaves.

Padding amplio.

Hover elegante.

Focus visible.

## Cards

Mucho espacio en blanco.

Bordes suaves.

Sombras ligeras.

Separación consistente.

## Inputs

Minimalistas.

Altura uniforme.

Estados claros.

Transiciones suaves.

## Modals

Entrada animada.

Fondo difuminado.

Escala ligera.

## Dropdowns

Animaciones fluidas.

Sin movimientos bruscos.

Toda la UI debe sentirse construida como un único sistema.
